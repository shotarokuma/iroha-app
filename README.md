# iroha-app
this project is coupon app using iroha api, which is sytsem of block chain by soramitu
### Install dependency
```
yarn install
```
### Run iroha server & db 
``` 
docker-compose -f docker/docker-compose.yaml up 
```
### Run client side
```
yarn client dev
```

### Run server side
```
yarn server dev
```

### Generate code for graphql
```
yarn graphql dev
```
